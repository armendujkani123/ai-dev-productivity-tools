# Security Policy

## Supported Versions

Security fixes will be applied to the latest version on the `main` branch.

## Reporting a Vulnerability

Please do not open public GitHub issues for security-sensitive reports.

Instead, contact the maintainer privately with:

- a description of the issue
- impact assessment
- reproduction steps
- suggested remediation, if available

The maintainer will acknowledge the report and work on a fix as quickly as possible.
